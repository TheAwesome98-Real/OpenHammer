Open source Goldsrc, Source, and Source 2 level editor for UNIX and Windows.

# To build
1. clone using git
2. run `make build` in src
3. run `./openhammer`
4. enjoy

# To contribute
1. fork this repo
2. make changes
3. verify they work (see 'To build')
4. run `make clean`
5. `git push` the changes to your repo
6. open pr
7. enjoy
